The PostgreSQL Server Installation Guide is a comprehensive guide to installing PostgreSQL via installers created by EnterpriseDB. In this guide you will find detailed information about:

- Hardware requirements and software prerequisites for PostgreSQL 13.0 installation on Windows.
- Graphical installation options available through the interactive setup wizard on Windows.
- Uninstalling PostgreSQL

## EDB Language Pack Installer

The EDB Language Pack Installer can be used with both PostgreSQL installations and Advanced Server installations. [Details are in the documentation](/epas/latest/language_pack/).
